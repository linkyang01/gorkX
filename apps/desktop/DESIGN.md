# gorkX UI design notes

## Current: light product shell

| Region | Color |
|--------|--------|
| App / chrome | `#f6f6f7` / sidebar `#f0f0f2` |
| Main chat | `#ffffff` |
| Selected row | soft gray wash `rgba(0,0,0,.09)` — **never pure white on light rail** |
| User bubble | soft gray `#ececee` (not black) |
| Primary / send | near-black `#111113` on light UI |

**No dark sidebar.** Keep airy, Codex-like light density.

Composer icons / chrome glyphs stay as-is.
